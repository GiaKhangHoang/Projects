# Task Manager (Java)

This is a console-based **Task Manager application** built in Java.  
It allows users to **add, view, and remove tasks** through a simple text-based menu interface.

## Features
- Add tasks with custom descriptions  
- View all tasks in a numbered list  
- Remove tasks by selecting their number  
- Menu-driven interface for easy navigation  
- Handles empty task lists and invalid inputs gracefully

## How to Run
1. Download or clone this repository  
2. Compile the Java file:
```bash
javac TaskManager.java
