# Grace Hopper Web Page

This is a simple HTML web page created as a learning project while practicing web development.  
It highlights the **life, achievements, and legacy of Grace Hopper**, a pioneering computer scientist and U.S. Navy Rear Admiral.

## Features
- Structured content using headings (`<h1>`, `<h2>`, `<h3>`) and paragraphs (`<p>`)  
- Emphasizes important information with **bold** and *italic* text  
- Includes an image of Grace Hopper and a link to an external website  
- Demonstrates basic HTML skills, including text formatting, images, and hyperlinks

## How to View
1. Download or clone this repository  
2. Open `index.html` in any web browser to view the page

## Key Learnings
- Practiced creating structured web content using HTML  
- Learned to format text for emphasis using `<b>` and `<i>`  
- Learned to include images and links to external resources  
- Gained experience in deploying simple web pages
